# Event Management System - Smart Campus

Full Stack Event Management Application built with:

- FastAPI (Backend)
- PostgreSQL (Database)
- React + Vite (Frontend)
- JWT Authentication

## Features

- User Registration & Login
- JWT Authentication
- Role-based Access (Admin/User)
- Create Events (Admin)
- Register for Events
- View My Registrations

## Tech Stack

Backend:
- FastAPI
- SQLAlchemy
- PostgreSQL

Frontend:
- React
- Axios
- React Router

## How to Run

### Backend

cd backend
uvicorn app.main:app --reload
